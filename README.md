# UnifiedSatAI Website

A professional website showcasing the **Unified Deep Learning Benchmark for Satellite Imaging** project, presenting research on satellite image restoration, classification, and evaluation using multiple datasets.

---

## Overview

This website serves as the primary platform for presenting the UnifiedSatAI project to researchers and educators. It provides:

- **Project Overview** - Clear explanation of the research scope and goals
- **Team Information** - Meet the researchers behind the project
- **Research Results** - Visual demonstrations of satellite image restoration and processing
- **Available Datasets** - Information on four key satellite imagery datasets:
  - AllClear (cloud removal)
  - PROBA-V (super-resolution)
  - SEN12MS-CR-TS (multispectral analysis)
  - WorldStrat (land cover classification)
- **Research Papers** - Downloadable PDFs of all project papers and reports
